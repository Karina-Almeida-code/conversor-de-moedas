## 🧠 Sobre o projeto

Este projeto permite converter valores entre diferentes moedas de forma rápida e intuitiva.

Foi desenvolvido como parte dos meus estudos em desenvolvimento web, com foco no uso do **JavaScript** para criar interatividade e lógica de conversão.

---

## ⚙️ Funcionalidades

- 💰 Inserção de valores  
- 🌍 Seleção de moedas (Real, Dólar, Euro)  
- 🔄 Conversão instantânea  
- 🧮 Cálculo realizado com JavaScript  
- 📱 Interface simples e responsiva  

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript
